How to setup the system

Here you can find the dependecies needed to run the snaps in an ubuntu 20.04 machine. These steps shold perform the same results in any ubuntu machine, anyway there could be the necessity to perform some other opertions to get the system ready to work! And of course you need a fully working bluetooth dongle installed on you machine!

1 - install mosquitto

This snap needs mosquitto to allow the excanges of data. The snap is configured to use an MQTT broker running on port 1883 with no protection. A suitable choice could be to use mosquitto as broker:

sudo snap install mosquitto

2 - install influxdb

This snap uses influxdb as time-series database. To istall it just use the following command:

sudo snap install influxdb-ijohnson

In this way the database starts working on port 8086 with no protection. The app creates a database named "room1" where all the data collected are stored.

3 - Install and setup Node Red

The snaps works using Node Red as frontend. It is very easy to setup node-red to be fully functional! just follow some easy step! First of all install Node Red:

sudo snap install node-red

Now open-up the node-red programming interface: node-red and install using the palette manager the following addons:

  • node-red-dashboard
  • node-red-contrib-influxdb

Now import the json file in node red or copy paste the content inside the restore window of node-red: node-red-flows
